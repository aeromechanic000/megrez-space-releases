# Megrez Space

A lightweight, modern desktop application designed for seamless file management and terminal-based workflows. Perfect for working with CLI agents.

Now this is a test version, you should run the following command in the terminal after installation (dragging Megrez Space.app to Applications).

```
xattr -cr /Applications/Megrez\ Space.app
```

For more information: [https://www.megrez.plus/megrez-space.html](https://www.megrez.plus/megrez-space.html).
